Fast and Efficient way to search for molecules

Instructions

To search molecules by molecule name:

Enter the molecule name in the search bar and press enter. The resulting molecule will show in the draw area and isomorphic results in the results area.

To search molecules by file name:

Drag and drop or select files into the drop area. Click add button then search button to see isomorphic results on the first molecule added.

Results

Results will be shown in the bottom or in the draw area.
